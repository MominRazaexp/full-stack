# Full-Stack React + Node/Express + SQLite Project

## Description
A full-stack project with React frontend, Node/Express backend, and SQLite integration.



## Installation
```bash
git clone https://github.com/MominRazaexp/full-stack.git
cd full-stack
npm install
```

## Usage
```bash
npm run dev
```
Visit `http://localhost:3000` in your browser.
